Final Project: Predictive Driving Behavior

Logan Bowers | Justin Bowers | Dean Longmire | Finli Hill

For the final project, the four of us decided on using machine learning, along with the data collected from the can bus of Dean's car, to classify driving behavior. To do this, we trained a machine learning model on driving that we sourced from kaggle.

Testing the model

Once the model was trained, we exported it into a binary file using the picle library in python. Here is the result of a script we wrote to simulate data flowing into the model.

Gathering Parts

The parts we gathered for this project included an OBD port adapter for CAN bus sniffing, an ESP32 for bluetooth connection, a buck converter, Arduino nano, MCP2515, a ttgo t-display, and we also 3D printed a box for the finished device to sit in.

Data flow of our device

Explanation: The OBD adapter allows us to plug in and sniff our data from the CAN bus of the car. That data is then reformatted into something that our machine learning model was trained on. The model then predicts the driving behavior that it is seeing from the Accelerator and Gyroscope x, y, and z data points.
